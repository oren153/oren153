# Oren Cohen

**Founder at [Converra](https://converra.ai)** — building the infrastructure to make AI agents improve themselves in production.

Converra is an agent optimization platform: simulation testing, regression gating, and continuous prompt improvement — so your agents get better without engineering upkeep.

## Currently shipping

**[agent-triage](https://github.com/converra/agent-triage)** — Open-source CLI + MCP server that diagnoses AI agent failures from conversation traces. Extracts behavioral policies from system prompts, evaluates with LLM-as-judge, and pinpoints exactly where agents break.

```bash
npx agent-triage analyze --traces conversations.json
```

Works with LangSmith, Langfuse, OpenTelemetry, Axiom, or raw JSON.
